===Introduction to Excess Return === 
Excess return is a measure of the performance of an investment relative to a benchmark. It is calculated by subtracting the benchmark return from the investment return. Excess return is also known as alpha, which is a measure of the performance of an investment relative to a benchmark. It is used to measure the performance of a portfolio manager or a fund manager.
===Calculating Excess Return === 
Excess return is calculated by subtracting the benchmark return from the investment return. For example, if an investment has a return of 10% and the benchmark has a return of 8%, then the excess return is 2%. The excess return is the difference between the investment return and the benchmark return.
===Examples of Excess Return === 
Excess return can be used to measure the performance of a portfolio manager or a fund manager. For example, if a portfolio manager has a return of 10% and the benchmark has a return of 8%, then the excess return is 2%. This means that the portfolio manager has outperformed the benchmark by 2%.
