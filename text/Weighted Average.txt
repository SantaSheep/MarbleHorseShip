===Introduction to Weighted Average===

Weighted average is a calculation used to determine the average of a set of numbers, where each number is assigned a different weight. This method of calculation is used in many different fields, including finance, economics, and statistics. Weighted average is a useful tool for analyzing data and making decisions based on the results.
===Calculation of Weighted Average===

The calculation of weighted average is relatively simple. To calculate the weighted average, the weights assigned to each number are multiplied by the corresponding number, and then the sum of these products is divided by the sum of the weights. For example, if there are three numbers with weights of 2, 4, and 6, the weighted average would be calculated as follows: (2 x number 1) + (4 x number 2) + (6 x number 3) / (2 + 4 + 6) = weighted average.

===Advantages and Disadvantages of Weighted Average===

This can be useful when analyzing data, as it allows for a more accurate representation of the data.
